# Adventure
This project requires Java 17 and is built using Gradle 7.5.1
# Project Tree
```bash

├── maps
├── sprites/
```

# Setting up and running
## Setup the project
```
gradle init
```

## Run the project
To run the application, please run the following command.
```
gradle run
```
